Kevcore Server Type (KCST):

Version 1:

Has KCoins and a few /trigger commands

Version 2:

Has all of the above with debug settings, updates command (optional: /trigger help), and a TPA system even if its disabled

Version 3:

Has all of the above with KCash and debug settings V2, and a login if the server is cracked

Version 4:

Has all of the above with a mining system even if its disabled, PKCODE (KC Private Code), and improved second delay using schedule

Version 5-snapshot1:

Has all of the above with a improved login (mostly from a plugin, authmereloaded) and commands with logins using PKCODE


Only Version 3 and above are supported and can get updates, while Version 4 and above will get regular updates

If a template version is a float type, it means there are additonal but not listed features and might not have them in later versions
