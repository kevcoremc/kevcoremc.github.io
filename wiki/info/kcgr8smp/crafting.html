Background:

DXC ASIC Miner (to increase the amount of DXC you get per second)

Note: You can only have 8 ASIC Miners, otherwise it will not work

To use the DXC ASIC Miner, make sure to have 8 or less and start mining using /trigger mine


Internet Block (To activate things that need internet like mining)

Note: It only sends Internet to players within 50 blocks of it

To use the Internet Block, drop it to the floor and activate the command block with a redstone signal

To break the Internet Block, crouch on top of it
