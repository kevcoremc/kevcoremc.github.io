What is an KCSMP server?

A KCSMP server is a vanilla server with command blocks to make the game more enjoyable.

A KCSMP server has to:

1. Be a vanilla server (No spigot, forge, etc)

2. Have custom commands using /trigger

3. Not just survival 

The server opens from 9AM - 9PM

Kevcore SMP was the second server to be made, and is still running to this day. Kevcore SMP is also known as KevSurvive or KCSMP.

KCSMP is the original server to be played on. It has a variety of fun stuff made using commands, such as custom enchantments, currency, and more all made using commands in Vanilla minecraft. Currency

Currently the only use for KCOINS (currency) is some items in the shop. Other than the Wood and ores, KCOINS are basically a collectible item you get from mining ores and killing mobs. Maybe in the future KCOINS will have a use.[2/10/2021 9PM]

How to play Kevcore SMP

Kevcore SMP is well, a SMP (Survival). No hacking or griefing, KCSMP is mostly a creation server. Bulid structures, houses, towns, and other stuff. KCSMP also has some minigames, such as sumo and a parkour. There are also custom commands, and by using the command /trigger help, you can see some custom commands you can use. /trigger help is not up-to-date, so you may find some commands that aren't in the /trigger help.[2/10/2021 9PM]

Secrets

There are lots of secrets. Some secrets come from commands. Some come from structures. Some are even items!

After the "No cheats, update", illegal items such as bedrock, command blocks, and barriers were removed. This is because before when there were town events, some illegal items like illegal enchantments were given. Now, they are removed from everyone, but you may find some illegal items still in chests.

Somewhere in the world has a chest of a OP diamond sword, having a sharpness 20. There is no exact location, but it is in a place near spawn[2/16/2021 1PM]

Storyline

If you don't know, Kevcore SMP was the second server to be made. Back then, there was a server called Kevcore Survival, a server similar to our Kevcore SMP today. But after Kevcore Survival experienced a huge corruption with no backups, the main sever was switched to KevSurvive. Kevcore SMP was created in hardcore mode, with no command blocks. Just vanilla. After a while, events started to happen. This is when hardcore mode was turned off. After town events are over, functions (datapacks) are created, replacing command blocks. If you were playing back then, you will know how annoying the new idlekick was. Town ranks are replaced by normal ranks (MVP, VIP), and Kevcore SMP became fun. KCoins had a use back then, which is the /trigger towntp command. It no longer exists today, as now we have /trigger tpa. Using /trigger towntp, you can easily travel back to your home town. Some people reported this as a easy TP cheat, so /trigger tpa was made.

Why did I get an error from the server?

Some reasons are that there are some problems with the server. Here is a list of the most common errors:

­

Error 100: There is nothing or no command line set (in beta)

Error 101: A command line is missing or not working properly

Error 102: The server is down or has low TPS (ticks per second)

Error 103: The command could not ban you either by the command not meeting the op permission level, coding failure, or Error 102

Error 104: The command could not kick you either by the command not meeting the op permission level, coding failure, or Error 102

Error 105: The command could not disconnect you either by the command not meeting the op permission level, coding failure, or Error 102

Error 106: undefined

Error 107: Can not get server TPS (the /trigger tps was removed)
